# ПРОФИБУД — сайт строительной компании

Реализация макета из `design/Estate Landing.dc.html` (формат Claude Design) как рабочего
статического сайта: чистый HTML + CSS + ванильный JS, без сборки и зависимостей.

## Запуск

Страницы подключают общие `header`/`footer`/`cta` через `fetch`, поэтому нужен
локальный HTTP-сервер (открытие `file://` напрямую не даст инъекции partials):

```bash
python3 -m http.server 8099
# затем открыть http://localhost:8099/index.html
```

## Структура

```
index.html                 Главная (Estate Landing) — эталонная страница
projects.html              Каталог: обзор категорий
projects-*.html            Категории проектов (dachnye / do70 / do120 / do180)
services.html              Услуги: обзор
services-*.html            Детали услуг (газосиликат / монолит / терраццо)
works.html                 Наши работы (галерея)

partials/                  Общие блоки (DRY между всеми страницами)
  header.html              Навигация + выпадающие меню
  footer.html              Подвал
  cta.html                 Блок «Оставить заявку»

assets/css/
  tokens.css               Дизайн-токены (цвета, шрифт, радиусы, брейкпоинты)
  base.css                 Reset, типографика, кнопки, утилиты
  components.css           Nav, карточки, форма, курсор, CTA
  responsive.css           Grid-утилиты + адаптив (1024 / 900 / 600)

assets/js/
  partials.js              Инъекция partials + подсветка активного пункта меню
  projects.js              Данные проектов + переиспользуемый рендер карточек
  form.js                  Сабмит заявки → состояние «Заявка принята»
  smooth-cursor.js         Кастомный курсор с пружинной физикой (Magic UI)

assets/img/                Ассеты из design/uploads/
```

## Дизайн-система

Все значения взяты из токенов макета (`assets/css/tokens.css`), не подбирались на глаз:
шрифт **Manrope**, палитра cream/charcoal/taupe/gold, радиус 24px, брейкпоинты 1024/900/600.

## Что перенесено из макета

- Все 9 секций главной, типографика и отступы 1:1.
- Анимации: подчёркивание навигации, выпадающие меню, scale фото на hover,
  hover-иконки карточек, кнопка отправки, кастомный smooth-курсор.
- Интерактив формы заявки (успешное состояние).
- Фото-плейсхолдеры (`<image-slot>` заменены стилизованными блоками — реальные фото
  подставляются позже).
